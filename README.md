# API para Cardápio Virtual

## Desenvolvimento local (Laravel)

Banco MySQL:

```sh
mysqld --console
```

Iniciar servidor de desenvolvimento:

```sh
php artisan serve --port=8001
```

# Deploy

Github Actions, commit na branch `main`
