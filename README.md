# API para Cardápio Virtual

## Desenvolvimento local

Iniciar servidor de desenvolvimento:

```
symfony server:start --no-tls --port=8001 -d
```

Parar servidor de desenvolvimento:
```
symfony server:stop
```

# Deploy

```
dep deploy
```
